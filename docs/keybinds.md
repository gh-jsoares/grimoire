# Keybinds

## Navigation

| Key | Action |
|-----|--------|
| `H` / `L` | Switch tabs |
| `1-9` | Jump to tab |
| `j` / `k` | Scroll |
| `g` / `G` | Top / bottom |
| `Ctrl+d` / `Ctrl+u` | Half-page scroll |
| `n` / `N` | Next / previous command |

## Actions

| Key | Action |
|-----|--------|
| `y` | Yank (copy) active command |
| `u` | Open link picker |
| `q` | Quit |

## Search

| Key | Action |
|-----|--------|
| `/` | Start search |
| `Tab` (in search) | Toggle filter/highlight mode |
| `Enter` | Confirm search |
| `Esc` | Clear search / cancel |
