# Installation

## From Source

```sh
go install github.com/gh-jsoares/grimoire/cmd/grimoire@latest
```

## With Make

```sh
git clone https://github.com/gh-jsoares/grimoire.git
cd grimoire
make install
```

## Homebrew

```sh
brew install gh-jsoares/tap/grimoire
```

## Shell Installer

Downloads the latest release binary for your platform:

```sh
curl -fsSL https://raw.githubusercontent.com/gh-jsoares/grimoire/main/install.sh | sh
```
